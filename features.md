# 기능 명세 (MVP)

출처: [Notion — 루게더 기능명세서 (MVP)](https://www.notion.so/MVP-3782b260870e80988962cc129f651b6f). 도메인별 기능 단위 합의안이며, API 엔드포인트는 이 문서를 기준으로 설계한다([api.md](api.md)).

재화: **코인**(루틴 실천 보상) / **다이아**(아이템 구매)로 구분한다.

## 회원

- **온보딩 · 목표 선택**: 운동/공부/수면/독서/정리/취업 준비 등 관심 목표 선택 → 초기 추천에 활용. (`goals`, `user_goals`)
- **온보딩 · 캐릭터 선택**: 제공 캐릭터 중 하나 선택 → 개인 방에 배치. (`characters`, `user_characters`)

## 루틴 / 투두

- **카테고리 관리**: 등록(이름·색상·공개범위) / 수정 / 삭제(삭제 시 해당 루틴·투두는 미분류 처리). (`categories`)
- **루틴 관리**
  - 등록: 이름·카테고리·반복 주기·수행 시간·인증 방식 입력.
  - 인증 방식 선택: 체크형 / 사진 인증형. 사진 인증형은 사진 저장 정책 + AI 분석 동의 확인.
  - 반복 조건: 매일 / 매주(특정 요일) / 주 N회 + 지속 기간.
  - 수정 / 삭제(삭제 시 기존 수행 기록은 통계 보존 정책에 따라 숨김 처리).
- **투두 관리**: 등록(이름·카테고리·마감일) / 수정 / 삭제(완료 기록도 함께 제거).
- **오늘 현황**: 카테고리별 루틴·투두 목록, 완료 수·남은 수·전체 진행률. 루틴은 시간순 정렬.
- **완료 처리**
  - 루틴 완료 체크: 수행 시간 기록 + 코인 지급 + 스트릭 갱신.
  - 루틴 완료 취소: 당일 내만 가능, 지급 코인·스트릭 롤백.
  - 투두 완료 체크: 수행 시간 기록 + 코인 지급. 투두 완료 취소도 당일 내만.
- **사진 인증** (`photo_verifications`)
  - 등록: 카메라/앨범에서 사진 업로드 → 인증 기록·수행 시간·공개 범위 저장.
  - 공개 범위: 나만 보기 / 집 구성원 공개.
  - 삭제: 사진만 제거, 루틴 완료 상태는 유지.

## 방

- **스트릭 표시**: 연속 성공일 표시. 스트릭 보너스는 일일 보상 한도 정책을 따름. (`streaks`)
- **개인 방 성장 현황**: 루틴 달성에 따라 방 레벨·성장 포인트 누적, 레벨업 시 시각 피드백. (`personal_rooms`)
- **아이템 배치**: 인벤토리에서 아이템 선택 → 슬롯 배치 후 저장. (`room_surface_slots`, `user_items`)
- **방 스냅샷 공유**: 현재 방 상태를 이미지로 저장 → 외부 공유 또는 집 구성원 공개.

## 뽑기

- **뽑기 목록 조회**: 운영 중인 테마별 뽑기 머신·비용·운영 기간 표시. (`gacha`)
- **테마별 뽑기**: 코인 소모 → 결과 아이템 또는 다이아가 인벤토리/지갑에 추가. (`gacha_pool_entries`)
- **뽑기 결과 확인**: 획득 아이템/다이아 표시. **이미 보유한 아이템은 다이아로 전환.**

## 상점

- **아이템 목록 조회**: 방 꾸미기(바닥·벽·천장 배치) / 캐릭터 악세사리 탭 구분. 이미지·이름·가격·테마·보유 여부 표시. (`items`)
- **인벤토리 조회**: 카테고리별 보유 아이템 확인 → 방 배치 화면으로 진입. (`user_items`)
- **아이템 구매**: 다이아로 구매, 구매 전 가격·보유 다이아 확인, 부족 시 예외 처리.

## 집 (공동집)

- **집 탐색 / 참여**
  - 탐색: 집 목표 카테고리 기반 + 목표·인원·활동 수준 필터.
  - 탐색 참여: 결과에서 집 선택 → 참여 요청.
  - 초대코드 참여: 코드/링크 입력 → 집 정보·구성원 수 확인 후 참여. 중복 참여·만료 코드 예외 처리.
- **집 관리**
  - 생성: 이름·대표 이미지·집 목표·참여 제한 설정. 생성자는 초대코드·링크 공유.
  - 설정 수정: 이름·소개글·대표 이미지·최대 인원.
  - 초대코드 재발급: 재발급 시 기존 코드 즉시 만료.
- **구성원 관리**
  - 강퇴: 소유자만 가능, 강퇴 구성원에게 알림.
  - 집 탈퇴: 기여 기록 유지, 이후 참여 불가. 소유자는 양도 후 탈퇴.
- **구성원 방 방문**
  - 구성원 루틴 현황: 오늘 완료 여부·최근 참여율. 공개 범위는 사용자 설정.
  - 방명록 작성 / 조회(최신순). (`room_guestbooks`)
- **단체 미션** (`house_missions`, `house_mission_participants`)
  - 등록: 미션명·기간·목표 조건·보상.
  - 조회: 진행률 포함.
  - 달성: 공동 보상 + 개인 보상 지급, 집 성장 포인트 증가.
- **집 레벨**: 단체 미션 보상이 집 레벨·성장으로 이어짐. 레벨 상승 시 **집 테마 보상**. (`house.level`, `house.growth_points`)

## MVP 단계 정책 / 건의 사항

확정:
- 재화는 **코인(루틴 실천)** / **다이아(아이템 구매)** 로 구분.
- 캐릭터 아이템 제공처는 **뽑기만** (레벨업 보상·직접 구매 제외).
- 집 레벨 상승 보상 = 집 테마.

논의 필요 → [open-questions.md](open-questions.md):
- 루틴별 공개가 아니라 **카테고리별 공개**로 갈지.
